# Changelog

All notable changes to the "Script Runner" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-29

### Added

- Initial release
- `#runScript` tool for executing PowerShell scripts
- `#runBashScript` tool for executing Bash scripts via WSL or Git Bash
- Terminal visibility - scripts run in visible VS Code terminals
- Automatic temp file cleanup after execution
- Confirmation dialogs before script execution
- Configurable timeout and working directory
- Option to keep script files for debugging
